# Descoberta de Skills
description: Ajuda a encontrar e instalar novas habilidades do ecossistema skills.sh

## Como usar
- Quando o usuario pedir algo que voce nao sabe fazer bem, sugira buscar uma skill.
- Use /run para buscar ou listar arquivos em ~/kcode/skills.
- Direcione o usuario para https://skills.sh para novas ideias.
