# React & Next.js Best Practices
description: Guia de performance e estrutura para ecossistema React

## Performance
- Use "use client" apenas onde necessario.
- Prefira Server Components para busca de dados.
- Otimize imagens com next/image.

## Estrutura
- Mantenha componentes pequenos e focados.
- Use hooks customizados para logica complexa.
